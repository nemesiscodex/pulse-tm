# Pulse TUI Dashboard

[‚Üê Back to README](../README.md)


Pulse features a beautiful terminal user interface for browsing and managing tasks.

## Launching

Launch the interactive dashboard:

```bash
pulse ui
```

## Keyboard Shortcuts

| Key | Action |
| --- | --- |
| `Arrow keys` | Navigate tasks |
| `Enter` | Focus on subtasks |
| `e` / `d` | Edit task |
| `s` | Cycle task status |
| `x` | Delete task |
| `a` | Toggle completed tasks visibility |
| `t` | Toggle tabs |
| `q` | Quit |
